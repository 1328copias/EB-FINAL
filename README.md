# EN BRUTO — Fondo limpio (negro o hormigón)

Por defecto el fondo es NEGRO.

Para usar el fondo de hormigón:
1) Abrí `index.html`
2) En `:root` cambiá:
   --bg-image: none;
   por:
   --bg-image: url("assets/bg-concrete.jpg");

Subí al repo:
- index.html
- assets/isologo.png
- assets/bg-concrete.jpg
